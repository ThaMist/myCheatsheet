My HTML and CSS cheatsheet

HTML semantic tags

tag description misc
<article> Defines independent, self-contained content Examples of where an article element can be used: Forum post Blog post Newspaper article
<aside> Defines content aside from the page content The aside content should be indirectly related to the surrounding content.
<details> Defines additional details that the user can view or hide use
<figcaption> Defines a caption for a use
<figure> Specifies self-contained content, like illustrations, diagrams, photos, code listings, etc. use
<footer> Defines a footer for a document or section A footer element typically contains: authorship information copyright information contact information sitemap back to top links related documents You can have several footer elements in one document.
<header> Specifies a header for a document or section A header element typically contains: one or more heading elements logo or icon authorship information
<main> Specifies the main content of a document use
<mark> Defines marked/highlighted text use
<nav> Defines navigation links use
<section> Defines a section in a document A web page could normally be split into sections for introduction, content, and contact information.
<summary> Defines a visible heading for a details element use
<time> Defines a date/time use
